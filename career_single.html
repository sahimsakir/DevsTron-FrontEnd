Software Engineer

We are searching for a software engineer who is proficient in any of the main programming languages, including C/C++, C#, Java, PHP, and Python.

Key Responsibilities

  • Design, create, test, and implement software in accordance with industry standards and coding methods.
  • Examine the specifications and comprehend the outputs.
  • By researching information requirements, system flow, data utilization, and work procedures, software solutions may be developed.
  • Create documentation, flowcharts, layouts, diagrams, charts, code comments, and clear code to support and illustrate solutions.
  • Reviewing code and design is something you may do after looking at existing software development projects.
  • Find innovative approaches and technology to enhance the project.
  • Participate in teamwork and make sure knowledge is transferred.

Skills Knowledge and Expertise

  • B.Sc or M.Sc in Computer Science/Engineering or equivalent.
  • Any significant programming language, such as C/C++, C#, Java, PHP, Python, etc., must be well-understood.
  • Any relational database, including Microsoft SQL Server, MySQL, Oracle, etc., must be familiar to you.
  • Must possess a solid understanding of SQL.
  • Any IDE, such as Microsoft Visual Studio, Eclipse, Netbeans, etc., must be acquainted to you.
  • Must be well-versed in the creation of object-oriented software.
  • It is preferable to have experience with client-side programming, such as JavaScript, AJAX, and jQuery.
  • It is preferred to have knowledge of ORM frameworks like Entity Framework and Hibernate.
  • English communication abilities, both oral and written, must be strong.
  • Must possess extremely good synthesis and analyzing skills.
  • Must be able to swiftly and independently pick up new technologies, tools, and approaches.
  • Must be proactive and capable of carrying out assignments successfully.

Experience Requirements

  • Fresh graduates are encouraged to apply.
